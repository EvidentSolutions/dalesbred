## Downloading

### Gradle

```kotlin
repositories {
    mavenCentral()
}

dependencies {
    implementation("org.dalesbred:dalesbred:<VERSION>")
}
```

### Maven

```xml
<dependency>
    <groupId>org.dalesbred</groupId>
    <artifactId>dalesbred</artifactId>
    <version><VERSION></version>
</dependency>
```

Replace `<VERSION>` with the latest version from [Maven Central](https://central.sonatype.com/artifact/org.dalesbred/dalesbred).
